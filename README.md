# Holo Blender Utilities

I've recently switched to Blender from Maya, and as someone who's used 3d software for a long time I have no doubt I'll build up an extensive library of functions for personal convenience.

Since these are primarily for my personal convenience, I feel no need to go out of my way to make them accessible for every user, but people who know the basics of Blender scripting should be able to make easy use of them.

I guarantee nothing about the practical use of these functions, nor do I guarantee I will maintain them.

## Features

Write now the list of features is only one, but more will get added as I need them:

- Toggle hide - toggles the hide value of the outliner selection so you don't have to unhide everything in your scene if you want to unhide one object with a keybind. It's hard to believe that Blender has simulations, beautiful cycles renders, video editing, a full sculpting suite --- but no toggle for hiding objects. It was surprisingly annoying to create.
